Understanding the Seven Layers of Computer Networking

OSI Layers

7. Application

Provides network services to users and applications.

6. Presentation

Translation, encryption and compression of data.

5. Session

Establishes and manages communication sessions.

4. Transport

Reliable delivery using TCP/UDP.

3. Network

Routing and logical addressing using IP.

2. Data Link

Physical addressing and frame transmission.

1. Physical

Transmission of bits through media.

Encapsulation Process

Data → Segment → Packet → Frame → Bits

WhatsApp Example

  • Application: User types a message.
  • Presentation: Data is encrypted.
  • Session: Session is maintained.
  • Transport: TCP ensures delivery.
  • Network: IP routes packets.
  • Data Link: Frames are sent using MAC addresses.
  • Physical: Bits travel through Wi‑Fi/mobile network.
